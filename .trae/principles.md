## 🔧 命令速查

### 🪟 Windows PowerShell 统一命令
| 操作类型     | 命令示例                                           | 图标 |
| ------------ | -------------------------------------------------- | ---- |
| **创建目录** | `New-Item -ItemType Directory -Path "路径" -Force` | 📁    |
| **创建文件** | `New-Item -ItemType File -Path "路径" -Force`      | 📄    |
| **删除文件** | `Remove-Item -Path "路径" -Force`                  | 🗑️    |
| **删除目录** | `Remove-Item -Path "路径" -Recurse -Force`         | 🗑️    |
| **复制文件** | `Copy-Item -Path "源" -Destination "目标" -Force`  | 📋    |

### 🚨 常见错误处理
| 错误场景     | 解决方案                            | 图标 |
| ------------ | ----------------------------------- | ---- |
| `mkdir` 失败 | 改用 `New-Item -ItemType Directory` | 🔧    |
| 路径包含空格 | 使用引号 `"路径"`                   | 📝    |
| 权限不足     | 添加 `-Force` 参数                  | ⚡    |

## 🔧 工具重构原则

### 🎯 功能重叠识别法

**📊 三维度分析法**
| 维度         | 分析内容         | 应用示例                     |
| ------------ | ---------------- | ---------------------------- |
| **功能边界** | 系统级 vs 项目级 | 性能监控归系统，脚手架归项目 |
| **用户场景** | 开发阶段划分     | 开始/开发中/部署前           |
| **代码扫描** | 快速识别重复     | 搜索相似功能关键词           |

**🎯 重构决策矩阵**
| 重叠类型     | 处理方式 | 示例                |
| ------------ | -------- | ------------------- |
| **完全重叠** | 直接合并 | 性能监控 + 系统优化 |
| **部分重叠** | 功能拆分 | 项目检查 → 项目助手 |
| **边界模糊** | 重新定义 | 交互增强 → 项目助手 |

### 🛠️ 重构实施步骤

**📋 渐进式重构流程**
```
阶段1：分析 → 识别重叠功能
阶段2：合并 → 创建统一工具
阶段3：验证 → 测试互斥互补
阶段4：清理 → 删除冗余文件
```

**✅ 验证标准**
- ✅ **互斥性**：工具间无功能重叠
- ✅ **互补性**：覆盖所有必要功能
- ✅ **兼容性**：100%跨平台支持
- ✅ **简洁性**：每个工具职责单一

## 🛠️ 跨平台工具集

### 🎯 5大核心工具架构

| 工具           | 职责边界     | 跨平台命令                                         | 使用场景 |
| -------------- | ------------ | -------------------------------------------------- | -------- |
| **系统增强器** | 系统级优化   | `python .trae/tools/system-enhancer.py --full`     | 环境优化 |
| **项目助手**   | 项目生命周期 | `python .trae/tools/project-assistant.py --report` | 项目管理 |
| **智能清理**   | 安全清理     | `python .trae/tools/smart-cleanup.py --analyze`    | 空间清理 |
| **智能提醒**   | 开发提醒     | `python .trae/tools/smart-reminder.py start`       | 关键节点 |
| **跨平台适配** | 命令统一     | 被其他工具内部调用                                 | 底层支持 |

### 🎪 快速使用指南

**🚀 开发工作流**
```bash
# 1. 开始新项目
python .trae/tools/project-assistant.py --report

# 2. 优化开发环境  
python .trae/tools/system-enhancer.py --full

# 3. 定期清理维护
python .trae/tools/smart-cleanup.py --analyze

# 4. 关键节点提醒
python .trae/tools/smart-reminder.py developing
```

**📊 重构成果**
- **从11个工具精简到5个核心**
- **100%跨平台兼容**
- **零功能重叠**
- **系统完备性验证通过**

### 🎓 学到的技能

**🔍 识别技巧**
1. **功能边界分析法**：按系统/项目级划分
2. **用户场景映射法**：按开发阶段划分
3. **代码功能扫描法**：快速识别重复

**🛠️ 重构技巧**
1. **职责单一原则**：每个工具只做一件事
2. **接口兼容性**：保持原有命令行接口
3. **渐进式重构**：先合并再优化最后清理

**✅ 验证方法**
1. **互斥性验证**：确保工具间无重叠
2. **互补性验证**：确保功能全覆盖
3. **兼容性验证**：确保跨平台正常工作
