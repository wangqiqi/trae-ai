# 🤖 项目管理智能体选择指南

## 🎯 一句话区分

- **项目经理** = "我有一个复杂项目，需要17个AI专家一起协作"
- **项目协调员** = "我想快速把需求变成技术方案和开发计划"

## 📊 选择决策表

| 你的情况 | 推荐智能体 | 使用示例 |
|---------|------------|----------|
| **新手第一次做项目** | 🎯 项目协调员 | `@项目协调员 我想做个博客网站` |
| **需要完整技术方案** | 🎯 项目协调员 | `@项目协调员 用React+Node做电商，给我详细方案` |
| **已有基础想扩展** | 🎯 项目协调员 | `@项目协调员 现有Vue项目如何添加支付功能` |
| **团队项目多人协作** | 👥 项目经理 | `@项目经理 启动一个AI图像识别项目，需要前后端配合` |
| **需要分工管理** | 👥 项目经理 | `@项目经理 分配任务：前端做界面，后端做API，测试写用例` |
| **项目进度跟踪** | 👥 项目经理 | `@项目经理 检查项目进度，看看各角色完成情况` |

## 🚀 实际使用对比

### 🎯 **项目协调员 - 个人项目**

**输入：**
```
@项目协调员 我想用Vue3做个任务管理应用
```

**输出：**
```
📋 项目方案：Vue3任务管理系统

🛠️ 技术栈：
- 前端：Vue3 + Vite + Element Plus
- 后端：FastAPI + SQLAlchemy  
- 数据库：SQLite
- 部署：Docker

📅 开发计划：
- 第1周：需求分析 + UI设计
- 第2周：前端开发
- 第3周：后端API + 集成
- 第4周：测试 + 部署

✨ 核心功能：
- 任务CRUD操作
- 用户登录/注册
- 任务分类和优先级
- 响应式设计
```

### 👥 **项目经理 - 团队协作**

**输入：**
```
@项目经理 启动AI图像识别项目，识别猫狗图片
```

**输出：**
```
🚀 项目启动：AI图像识别系统

👥 团队分工：
- @产品经理：需求文档 + 用户故事
- @系统架构师：微服务架构设计
- @Python-AI工程师：模型训练 + API开发
- @Vue工程师：管理后台界面
- @测试工程师：测试用例 + 性能测试

📊 项目看板：
- 阶段1：需求确认 (产品经理)
- 阶段2：架构设计 (系统架构师)
- 阶段3：模型开发 (Python-AI工程师)
- 阶段4：前端开发 (Vue工程师)
- 阶段5：集成测试 (测试工程师)

⏰ 时间线：6周完成
```

## 💡 使用技巧

### 🔄 **智能切换**

1. **先用协调员** → 快速启动项目
2. **项目变大** → 切换到项目经理进行团队管理
3. **遇到问题** → 协调员提供技术方案，项目经理分配解决任务

### 📈 **成长路径**

```
新手用户 👉 项目协调员 👉 熟悉后 👉 项目经理
简单项目 👉 技术方案 👉 复杂后 👉 团队协作
```
