# 🤝 人机协作规范

## 📋 规范目的

本规范定义了人类用户与18人AI超级团队之间的**标准化沟通协议**，确保每次交互都高效、清晰、可追溯。

---

## 🎯 核心沟通原则（5步黄金流程）

### **第1步：深度理解** 🧠
- **AI行为**：主动分析用户自然语言的**意图、场景、约束条件**
- **输出要求**：复述用户需求，确认理解准确性
- **示例**：
  > 用户："我想做一个小程序"
  > AI："我理解您想创建一个微信小程序项目，主要功能需求是...技术栈建议是...时间预期是..."

### **第2步：思考展示** 💭
- **AI行为**：**透明化思考过程**，展示分析逻辑
- **输出要求**：使用思维导图格式展示决策树
- **示例**：
  ```
  思考过程：
  ├─ 项目类型识别：电商小程序
  ├─ 技术栈选择：uni-app + FastAPI
  ├─ 团队配置：项目经理→uni-app工程师+FastAPI工程师+产品经理
  └─ 时间评估：3周标准开发周期
  ```

### **第3步：规划方案** 📊
- **AI行为**：提供**2-3种可选方案**，包含利弊分析
- **输出要求**：每个方案包含：技术方案、团队配置、时间成本、风险点
- **模板**：
  | 方案 | 技术栈 | 团队配置 | 时间 | 风险 | 推荐指数 |
  |---|---|---|---|---|---|
  | A | uni-app+FastAPI | 3人团队 | 3周 | 低 | ⭐⭐⭐⭐⭐ |
  | B | 原生开发 | 5人团队 | 6周 | 中 | ⭐⭐⭐ |

### **第4步：等待确认** ✅
- **AI行为**：**明确询问用户选择**，不擅自执行
- **标准话术**："请确认您选择哪个方案？或有其他需求调整？"
- **等待信号**：用户明确回复"确认"、"同意"、"开始"等关键词

### **第5步：总结交付** 🎉
- **AI行为**：执行完成后**系统总结**
- **输出要求**：包含：执行结果、关键决策、后续建议、学习要点
- **模板**：
  ```
  ✅ 项目完成总结
  ├─ 实际执行：uni-app电商小程序
  ├─ 关键成果：商品展示+购物车+微信支付
  ├─ 团队表现：项目经理协调高效，提前2天完成
  └─ 后续建议：可增加用户评价功能
  ```

## 🗣️ 自然语言交互标准

### **高质量需求描述模板**
```
我想做【项目类型】，
主要功能包括【功能列表】，
目标用户是【用户群体】，
预期时间是【时间要求】，
技术偏好是【技术栈倾向】，
其他要求【特殊需求】。
```

### **AI理解确认机制**
- **必问要素**：项目类型、功能范围、技术偏好、时间预期、质量要求
- **澄清话术**："关于【某要素】，我理解是...是否正确？"
- **边界确认**："哪些功能不在本次开发范围内？"

## 📖 思考过程展示规范

### **决策树格式**
```
决策分析：
├─ 需求复杂度：低/中/高
├─ 技术可行性：✅可行/⚠️有风险/❌不可行
├─ 团队匹配度：完美匹配/需要调整/建议外包
├─ 时间合理性：宽松/合理/紧张
└─ 推荐方案：【具体建议】
```

### **风险评估矩阵**
| 风险类型 | 概率 | 影响 | 应对措施 | 责任人 |
|---|---|---|---|---|
| 技术风险 | 低 | 中 | 技术预研 | 系统架构师 |
| 进度风险 | 中 | 高 | 增加人手 | 项目经理 |
| 需求变更 | 高 | 中 | 敏捷开发 | 产品经理 |

## 📊 质量保证机制

### **交互质量评分**
- **理解准确度**：用户确认无歧义（5分制）
- **方案完整性**：覆盖所有需求要素（5分制）
- **执行效率**：无重复沟通（5分制）
- **用户满意度**：最终交付评价（5分制）

### **持续改进循环**
1. **每次交互后**：收集用户反馈
2. **每周总结**：优化沟通模板
3. **每月更新**：完善协作规范
4. **季度回顾**：升级整个协作体系

## 🎯 快速上手指南

### **新用户30秒上手**
1. **说出需求**：用自然语言描述项目
2. **等待分析**：AI展示思考过程
3. **选择方案**：从2-3个选项中确认
4. **坐享其成**：项目经理统一协调完成

### **常用话术模板**
- **启动项目**："项目经理，请启动【项目类型】，功能包括【功能列表】"
- **调整需求**："项目经理，需求有变化：【具体变更】"
- **查询进度**："项目经理，当前项目进展如何？"
- **技术咨询**："系统架构师，这个技术方案可行吗？"
