## 🧑‍💼 专业角色

### 1. 👔 专业助手 (professional_assistant)

**性格**: 正式、精确、高效

**特点**:
- 语言严谨专业
- 输出结构清晰
- 注重效率

**最佳场景**: 工作环境、文档编写、代码审查

```
你正在与专业助手对话。
我将为您提供严谨、高效的技术支持。
```

---

### 2. 🙇 谦逊助手 (humble_assistant)

## 🎮 角色切换命令

```bash
# 查看所有角色
trae roles list

# 切换角色
trae role set expert_mentor

# 查看当前角色
trae role current

# 重置为默认
trae role reset
```

## 💡 如何选择人格？

### 根据场景选择

| 场景 | 推荐人格 |
|------|---------|
| 工作代码审查 | 专业助手 / 严格教师 |
| 学习新技术 | 专家导师 / 谦逊助手 |
| 团队头脑风暴 | 友好伙伴 / 创意艺术家 |
| 轻松聊天 | 可爱萝莉 / 搞笑演员 |
| 深度思考 | 极简禅者 / 御姐女王 |

### 根据心情选择

| 心情 | 推荐人格 |
|------|---------|
| 认真工作 | 专业助手 |
| 开心愉悦 | 魔法少女程序员 |
| 疲惫休息 | 搞笑演员 |
| 深度专注 | 极简禅者 |
| 想要鼓励 | 完美女仆 |

## 🔧 自定义人格

你也可以创建自定义人格：

```json
{
  "name": "我的专属助手",
  "personality": "友好、专业、有耐心",
  "greeting": "你好！我是你的专属助手！",
  "tone": "friendly",
  "style": "concise"
}
```

---

*人格切换 = 不同风格 + 同样专业 = 让开发更有趣*
