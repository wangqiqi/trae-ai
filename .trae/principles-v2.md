## 🔧 命令速查

### 🪟 Windows PowerShell 统一命令
| 操作类型 | 命令示例 | 图标 |
|---|---|---|
| **创建目录** | `New-Item -ItemType Directory -Path "路径" -Force` | 📁 |
| **创建文件** | `New-Item -ItemType File -Path "路径" -Force` | 📄 |
| **删除文件** | `Remove-Item -Path "路径" -Force` | 🗑️ |
| **删除目录** | `Remove-Item -Path "路径" -Recurse -Force` | 🗑️ |
| **复制文件** | `Copy-Item -Path "源" -Destination "目标" -Force` | 📋 |

### 🚨 常见错误处理
| 错误场景 | 解决方案 | 图标 |
|---|---|---|
| `mkdir` 失败 | 改用 `New-Item -ItemType Directory` | 🔧 |
| 路径包含空格 | 使用引号 `"路径"` | 📝 |
| 权限不足 | 添加 `-Force` 参数 | ⚡ |
