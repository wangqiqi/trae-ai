# Trae IDE AI助手使用指南

## 🎯 直接在对话框中使用

### 使用方法：
**直接在Trae对话框输入，无需任何命令！**

## 🔥 三种交互模式

### 1️⃣ @智能体调用（精确控制）
```
@产品经理 我想做一个任务管理应用，需要用户注册登录
```

### 2️⃣ #项目类型（快速创建）  
```
#todo 创建一个Vue3的任务管理应用，支持分类和优先级
```

### 3️⃣ 自然语言（AI自动分配）
```
我想做一个电商网站，支持微信支付，大概3周完成
```

## 🎭 18个智能体角色

| 智能体 | 用途 | 示例 |
|--------|------|------|
| **@产品经理** | 需求分析、功能规划 | `@产品经理 分析这个需求` |
| **@系统架构师** | 技术选型、架构设计 | `@系统架构师 Vue3+FastAPI架构` |
| **@Vue工程师** | Vue.js前端开发 | `@Vue工程师 创建任务列表页面` |
| **@React工程师** | React前端开发 | `@React工程师 设计商品展示页` |
| **@Python工程师** | FastAPI后端开发 | `@Python工程师 创建CRUD API` |
| **@Node工程师** | Node.js后端开发 | `@Node工程师 设计用户认证` |
| **@AI工程师** | 机器学习集成 | `@AI工程师 集成图像识别` |
| **@测试工程师** | 测试用例设计 | `@测试工程师 为登录功能写测试` |
| **@DevOps工程师** | 部署和运维 | `@DevOps工程师 Docker部署方案` |
| **@项目经理** | 项目整体管理 | `@项目经理 启动电商项目` |
