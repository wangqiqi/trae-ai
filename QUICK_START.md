### 场景1：Python AI项目（图像分类）

```bash

cd e:\study\learn_trae

# 2. 复制AI模板到新项目目录
mkdir my-ai-project
cd my-ai-project
cp ..\.trae\templates\ai-python.json project-template.json
```

## 🆘 常见问题

**Q: 如何选择技术栈？**
A: 查看`.trae/config/team-config.json`中的优先级配置：
- 前端：Vue > React > Angular
- 后端：FastAPI > Node.js
- AI：Python AI > C++部署

**Q: 智能体输出质量不好怎么办？**
A: 
1. 检查需求描述是否清晰
2. 提供更多上下文信息
3. 分步骤调用（先需求分析，再设计，最后实现）

**Q: 如何保存项目特定的配置？**
A: 每个项目创建时复制`.trae`目录，然后按需修改配置文件
