## 🆘 常见问题

**Q: 如何选择技术栈？**
A: 
- **MVP模式**：直接指定MVP模板，1小时启动
- **项目经理模式**：无需选择，项目经理会根据需求自动选择最优技术栈
- **手动模式**：查看`.trae/config/team-config.json`中的优先级配置：
  - 前端：Vue > React > Angular
  - 后端：FastAPI > Node.js
  - AI：Python AI > C++部署

**Q: 项目经理如何工作？**
A: 
1. **自动分析需求** - 理解项目类型和复杂度
2. **智能分配角色** - 根据需求选择最适合的专家组合
3. **制定开发计划** - 创建详细的时间线和里程碑
4. **统一协调进度** - 管理各阶段Review和质量控制
5. **风险预警** - 识别潜在问题并提供解决方案

**Q: 智能体输出质量不好怎么办？**
A: 
1. **项目经理模式**：项目经理会统一优化需求描述和上下文
2. **单角色模式**：
   - 检查需求描述是否清晰
   - 提供更多上下文信息
   - 分步骤调用（先需求分析，再设计，最后实现）

**Q: 如何保存项目特定的配置？**
A: 
- **项目经理模式**：项目完成后，项目经理会生成项目专属配置
- **手动模式**：每个项目创建时复制`.trae`目录，然后按需修改配置文件

**Q: 项目经理支持哪些项目类型？**
A: 项目经理支持以下项目类型的自动配置：
- **MVP极速**：Vue3/FastAPI/SQLite 1-2周交付
- **Web应用**：Vue/React/Angular + FastAPI/Node.js
- **移动App**：Flutter/uni-app + Go/Rust后端
- **小程序**：uni-app多端 + FastAPI后端
- **AI系统**：Python AI + C++部署 + 前端展示
- **微服务**：Go微服务 + React前端
