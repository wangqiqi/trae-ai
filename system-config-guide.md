文件位置：`C:\Users\[你的用户名]\.trae-cn\user_rules.md`

### 📋 步骤2：添加核心原则（复制粘贴）

#### 🎯 时间信息原则（必须添加）
```markdown
## 🕐 时间信息原则 (SYSTEM-TIME)
- **强制要求**：每次对话获取当前准确时间
- **自动更新**：所有文档版权年份为当前年份
- **时间戳**：所有代码提交包含准确时间信息
```

#### 🎯 项目结构标准化
```markdown
## 📁 项目结构规范 (SYSTEM-STRUCTURE)

```


#### 🎯 智能体调用规范
```markdown
## 🤖 智能体使用规范 (SYSTEM-AGENTS)
```

## 🔧 一键应用方法

### 方法A：使用脚本（推荐）
1. 右键以管理员身份运行：`apply-system-config.bat`
2. 自动完成所有配置

### 方法B：手动复制
1. 打开`C:\Users\[用户名]\.trae-cn\user_rules.md`
2. 复制上面的核心原则内容
3. 粘贴到文件末尾
4. 保存文件

### 方法C：备份恢复
```bash
# 备份当前配置
copy "%USERPROFILE%\.trae-cn\user_rules.md" "%USERPROFILE%\.trae-cn\user_rules.md.backup"

# 使用模板（手动复制system-level-user-rules-template.md内容）
```
